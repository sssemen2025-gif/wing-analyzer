## 📋 О проекте

Приложение предназначено для аэродинамического анализа крыла летательного аппарата.  
Исходные данные — **два STEP-файла** (верхняя и нижняя поверхности крыла).

Программа автоматически нарезает сечения, строит профили и выполняет гибридную аппроксимацию.

---

## 🚀 Основные возможности

- Загрузка двух STEP-файлов (верх/низ)
- Автоматическая и ручная нарезка сечений по Z
- Гибридная аппроксимация **CST + PCHIP**
- 2D и 3D визуализация
- Экспорт в авиационный формат **.056**
- Удобный GUI на PySide6 (тёмная тема)

\## 📋 О проекте



Приложение предназначено для аэродинамического анализа крыла летательного аппарата. Исходные данные — \*\*два STEP-файла\*\* (верхняя и нижняя поверхности крыла). Программа автоматически нарезает сечения вдоль оси Z, строит профили и позволяет:



\- Визуализировать сечения в 2D (абсолютные и относительные координаты)

\- Выполнять \*\*гибридную аппроксимацию\*\* профилей: CST для носа + PCHIP для хвоста

\- Строить 3D-модель крыла с подсветкой кромок (Matplotlib + PyVista)

\- Экспортировать результаты в авиационный формат \*\*.056\*\*



Проект разработан \*\*для демонстрации инженерных компетенций\*\* (пет-проект).

\## 🚀 Возможности



| Функция | Описание |

|---------|----------|

| 📂 \*\*Загрузка STEP\*\* | Чтение двух STEP-файлов поверхностей через pythonocc-core |

| ✂️ \*\*Нарезка сечений\*\* | Автоматическая (равномерно по Z) или ручная (указать координаты) |

| 📐 \*\*Аппроксимация CST+PCHIP\*\* | Гибридный метод: полиномы Бернштейна для носика (0–15% хорды) и кусочно-кубическая интерполяция для хвоста |

| 🔄 \*\*Нормализация\*\* | Поворот и масштабирование сечений относительно хорды |

| 📊 \*\*2D визуализация\*\* | Просмотр сечений: сырые точки, контур, замкнутость |

| 🌐 \*\*3D визуализация\*\* | Объёмная модель крыла с передней и задней кромками (Matplotlib 3D + PyVista) |

| 💾 \*\*Экспорт .056\*\* | Формат для аэродинамических расчётов (координаты, TRAIL, SLOPT, XSING) |

| ⚡ \*\*Многопоточность\*\* | Обработка в QThread, не блокирует интерфейс |

| 🎨 \*\*Тёмная тема\*\* | Стиль Fusion из коробки |
